# RPV51 — продающий сайт автосервиса (Ярославль)

Современный конверсионный лендинг на чистом HTML/CSS/JS. Без сборки и зависимостей — открывается в любом браузере и размещается на любом хостинге. Заполнен **реальными данными** автосервиса (rpv51.ru, Яндекс.Карты, Zoon).

## Структура

```
rpv51-yaroslavl/
├── index.html      — разметка и контент
├── styles.css      — палитра «Графит + Янтарь», адаптив
├── script.js       — модалка записи, лайтбокс, маска телефона, формы, меню
├── assets/         — реальные фото сервиса (hero.jpg, work1–work6.jpg)
└── README.md
```

## Как посмотреть

Откройте `index.html` двойным кликом, либо локальный сервер:
```bash
cd rpv51-yaroslavl && python3 -m http.server 8000   # http://localhost:8000
```

## Цветовая палитра (светлая)

Светлая тема: белый фон `#ffffff` / мягкий `#f4f7fb` + тёплый янтарный акцент (`#c8800a` для текста, `#f5a623` в заливках). Тёмный читаемый текст, мягкие тени для глубины. Меняется в `:root` файла `styles.css`.

## ⚠️ Что нужно подключить (важно)

Форма сейчас **валидируется, но никуда не отправляет** заявку (пишет в консоль). В `script.js` есть комментарий «ПОДКЛЮЧИТЕ ОТПРАВКУ». Варианты:
- **E-mail без бэкенда** — Formspree / Getform — рекомендуется;
- **CRM** (Bitrix24 и др.) — POST на вебхук лида;
- **ВКонтакте** — приём заявок в сообщения сообщества.

Подскажу с любым — скажите, куда слать заявки.
