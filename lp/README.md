# RPV51 — рекламный лендинг (отдельный домен)

Одностраничный лендинг под платный трафик (Яндекс.Директ + VK Ads). Отдельный проект от основного сайта `rpv51-yaroslavl/` — рассчитан на размещение на **отдельном домене** (напр. `rpv51-zayavka.ru`, `авто-ремонт-ярославль.рф` и т.п.). Чистый HTML/CSS/JS, без сборки.

## Файлы
```
rpv51-lending/
├── index.html   — лендинг (hero+квиз, услуги, отзывы, ВК-блок, FAQ, CTA, контакты)
├── styles.css   — светлая палитра + янтарный акцент, адаптив, квиз, sticky-панель
├── script.js    — квиз-калькулятор, лиды в ВК и по телефону, аналитика (конфиг-гейтинг)
├── assets/      — реальные фото сервиса
└── README.md
```

## Запуск
```bash
cd rpv51-lending && python3 -m http.server 4174   # http://localhost:4174
```

## Конверсионная механика
- **Квиз-калькулятор** в первом экране (4 шага → захват телефона) — главный лид-магнит под рекламу
- Заявки уходят в **ВКонтакте** и по телефону
- Липкая мобильная панель: **Звонок · ВК · Записаться**
- Плавающая кнопка ВК, модалка записи, лайтбокс галереи
- Оффер на «честности» (главная незанятая ниша среди конкурентов Ярославля): «чинят только то, что реально сломалось, смета до работ»
- Доказательства: 5,0★ / 73 отзыва / «Хорошее место 2026», живой виджет Яндекс.Карт, реальные фото

## ⚙️ Что заполнить перед запуском рекламы (в `script.js`, объект `CONFIG`)
| Поле | Зачем |
|---|---|
| `metrikaId` | номер счётчика Яндекс.Метрики (цели: `lead`, `call`, `vk`, `quiz_lead`) |
| `vkPixelId` | пиксель VK Ads (top.mail.ru) для оптимизации и ретаргета |
| `vkGroupId` | числовой ID сообщества (для живого ВК-виджета; узнать: regtools.ru) |

Без этих ID аналитика и ВК-виджет просто не подключаются — лендинг работает, ничего не падает.

## ⚠️ Отправка заявок
Формы и квиз валидируются и ведут в ВК, но серверной отправки нет (маркер «ПОДКЛЮЧИТЕ ОТПРАВКУ» в `script.js`). Варианты без бэкенда: **Formspree / Getform** (на e-mail) или вебхук CRM. Заявки в ВК идут через `vk.me/rpv51` (нужно включить «Сообщения сообщества»).

## Публикация
Netlify / Vercel / Cloudflare Pages (перетащить папку) или хостинг по FTP. Привязать отдельный домен и настроить HTTPS.

## Рекламный кит
Заголовки A/B, объявления Яндекс.Директ и VK Ads, ключи, минус-слова, аудитории и UTM-схема — в файле `AD-KIT.md`.
